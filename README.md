Item 01 - Formulário de Newsletter não aceita submissão
Tipo: Correção
Classificação: Utilidade
Prioridade: Alta
Descrição: Mesmo com todos os campos preenchidos de forma válida ou com somente os campos obrigatórios preenchidos, o formulário de newsletter não é submetido. É exigido " informar base legal" para o envio, entretanto não há esse campo no formulário.
Página(s) relacionada(s): Site
